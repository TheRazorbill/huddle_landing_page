## My process

### Built with

- Marcação semântica HTML5
- Propriedades de CSS3
- **Flexbox** (para o layout principal de duas colunas e alinhamento de ícones)
- **Google Fonts** (Poppins e Open Sans)
- **Font Awesome** (para os ícones de redes sociais)
- Fluxo de trabalho *Desktop-first*, com uma *media query* para responsividade mobile.

### What I learned

Este projeto foi uma ótima prática para consolidar o uso de **Flexbox** para criar um layout responsivo de duas colunas que se adapta a telas menores.

Também foi uma boa prática para integrar bibliotecas de terceiros de forma eficiente, como Google Fonts para a tipografia e Font Awesome para os ícones, garantindo que o resultado final ficasse fiel ao design.

Um pequeno, mas importante, detalhe foi o uso de `aria-label` nos links das redes sociais para melhorar a acessibilidade:

```html
<a href="#" aria-label="Facebook">
  <i class="fab fa-facebook-f"></i>
</a>
```
